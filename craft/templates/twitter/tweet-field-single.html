Single Tweet field

Configuration

  1. Install the Twitter plugin.
  2. Create a Tweet field called Single Tweet (singleTweet).
  3. Create a global called Twitter Example (twitterDemo).
  4. Add the “Single Tweet” to the Twitter Example global fields under the “Field Layout” tab.
  5. Go to CP → Globals → Twitter Demo and add some tweet URLs.
